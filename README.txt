7ми сегментные индикаторы вольметра работают на основе мультиплексирования.
По сигналу модуля сравнения таймера TIM3 происходит начало пребразования 
напряжения ADC1. После каждого преобразования ADC генерирует запрос DMA, 
вследствие которого DMA2 отправляет измеренное значение в ранее опредёлённый 
массив. Массив используется для вычисления среднего значения. Когда массив 
заполняется, вызывается прерывание DMA2 по окончанию передачи данных первой
транзакции. Впрерывании меняется включается следующий индиктор. В каждый момент 
времени включён только один индикатор, но так как частота переключения высокая,
то длячеловека оба индикатора горят одновременно - это и есть мультиплексирование.
DMA2 работает в режиме circular, поэтому после окончания первой транзакции 
автоматически включается вторая до заполнения массива и т.д.
